# Decision Log — Léognan Événements

## 2026-05-23

### Contexte établi
- Projet porté par l'opposition municipale de Léognan, équipe ~5 personnes
- Problème : communication municipale défaillante sur les événements locaux
- La mairie envisage un appel d'offres pour une app officielle mais sans calendrier

### Décisions clés
- **Nom provisoire** : "Léognan Événements" [HYPOTHÈSE — à valider avec l'équipe]
- **Plateformes MVP** : iOS + Android
- **MVP = démo fonctionnelle** avec données placeholder, pas de saisie utilisateur
- **Backoffice** : reporté en V2, avec flux de modération (soumission → validation → publication)
- **Notifications push + publication FB/Insta** : V2
- **Modération** : assurée par les membres de l'équipe citoyenne (~5 personnes)
- **Structure événement** : titre, description, image, date début, date fin (ou journée entière), catégorie, lien optionnel

### Contraintes confirmées (2026-05-23)
- Développement solo (Camil + Claude Code), budget nul
- Hébergement MVP : Vercel ou perso, gratuit — scalabilité V2/V3
- Stack à privilégier : simple, maintenable seul, compatible Vercel

### Hypothèses confirmées (2026-05-23)
- Nom plateforme : **Ma Ville Agenda** — simple, français, générique. Le déploiement Léognan s'appellera "Léognan Agenda" (nom de l'instance), la plateforme reste "Ma Ville Agenda".
- Cible ~500 téléchargements en 3 mois : réaliste selon l'équipe
- FB/Insta : confirmé en V2
- Vision marque blanche : confirmée — logos et CSS paramétrables par commune, déploiement multi-tenant en V3+. Léognan est le premier déploiement de référence.
